# AI Business Assistant Bot 

Интеллектуальный Telegram-бот для автоматизации консультаций в бизнесе.

### Стек технологий:
- **Language:** Python 3.11+
- **Framework:** Aiogram 3.x (Asynchronous)
- **AI:** OpenAI API (GPT-4)
- **Deployment:** Docker, Ubuntu VPS

### Что реализовано:
- Асинхронная обработка запросов пользователей.
- Интеграция с GPT-4 для генерации экспертных ответов.
- Система логирования ошибок.
- Чистая архитектура, готовая к масштабированию (RAG, Vector DB).
